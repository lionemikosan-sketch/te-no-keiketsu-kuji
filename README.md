# 手の経穴くじ 🎴

鍼灸学生向けの **手の経穴 取穴練習アプリ**。全25穴からランダムに9枚を引き、くじ形式で取穴を確認できます。単一HTMLファイル・外部依存ゼロ、ブラウザで開くだけで動きます。

## ✨ 特長

- **9枚のカードくじ**（3×3グリッド） … 毎回 全25穴から重複なしで9枚を抽出
- **各カード独立の3状態フリップ**
  1. 裏面
  2. 1タップ目：**経穴名**だけを大きく表示（まず取穴を考える）
  3. 2タップ目：**要穴／刺激方法／角度／深さ／アドバイス**で答え合わせ
  4. 3タップ目：裏面に戻る
- **再シャッフル** … 9枚を引き直し、全カードを裏面にリセット（ディール演出付き）
- **全カードを裏に戻す** … 同じ顔ぶれのまま表示だけリセット
- 黒ベース＋シアン／ゴールドの上品なデザイン、滑らかなフリップと発光演出
- PC優先・レスポンシブ対応・タッチ操作対応・キーボード操作対応（Tab → Enter / Space）

## 🚀 使い方

1. `index.html` をブラウザで開くだけ（インストール・サーバー不要）。
2. カードをタップして 経穴名 → 詳細 → 裏面 と切り替えながら取穴を練習。

> 公開サイト（GitHub Pages）でもそのまま遊べます。

## 🛠 データの編集

経穴データは `index.html` 内 `<script>` の `POINTS` 配列にまとまっています。穴の追加・修正はこの配列を編集するだけで反映されます。
