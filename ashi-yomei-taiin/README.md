# 経穴くじ

足の陽明胃経・足の太陰脾経の、膝周囲から末端にある18の要穴を重複なしのカードで学ぶWebアプリです。

## 使い方

1. 各経穴1枚のカードから1枚選ぶと経穴名が表示されます。
2. 同じカードをもう一度タップすると要穴分類が表示されます。
3. 「まぜ直す」で新しいカード配置になります。

胃経のみ・脾経のみの出題にも切り替えられます。

## 収録範囲

- 足の陽明胃経：梁丘、足三里、上巨虚、下巨虚、豊隆、解渓、衝陽、陥谷、内庭、厲兌
- 足の太陰脾経：隠白、大都、太白、公孫、商丘、三陰交、地機、陰陵泉
